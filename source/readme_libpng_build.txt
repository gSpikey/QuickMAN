Instructions for rebuilding libpng_writeonly_min.lib:

1. Go to http://sourceforge.net/projects/libpng/ and download the 
source code for libpng and zlib.

2. Run libpng_gather.bat to copy the required files to your 
working directory (change the source and destination directories 
in the batch file if necessary).

3. Load libpng_writeonly_min.vcproj into Visual Studio and build.
